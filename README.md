About
=====

The simplest Web Application with Spring Boot

Requirement 
============

* `JDK 1.8 or later`
* `Gradle 4`
* `IDE IntelliJ community edition or professional edition`

Run
===

`./gradlew bootRun` or from IDE's run command (Grandle task)

Checks
=====
* `http://localhost:8080`
* `http://localhost:8080/actuator/health`
